# Blocpress

Lightweight document template/rendering engine. Takes LibreOffice Writer templates (ODT) plus JSON data and produces final documents (ODT/PDF/RTF).

## Modules

- [blocpress-renderer](blocpress-renderer/summary.html) -- Core library
- [blocpress-server](blocpress-server/summary.html) -- Quarkus REST API
