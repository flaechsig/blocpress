How It Works

Send JSON data to the Blocpress server. The engine merges data into LibreOffice ODT templates and returns a generated PDF. 

curl -X POST http://localhost:8080/render \
  -H "Content-Type: application/json" \
  -d '{"template":"contract.odt","data":{"name":"John Doe"}}'
