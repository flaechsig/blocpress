Quickstart

Start the server, download a sample template with matching JSON data, and generate your first PDF in seconds.

1. Run the container

docker run --name blocpress-render -d -p 8080:8080 flaechsig/blocpress-render

The image ships with a built-in dev key and issuer. To use your own JWT configuration, override via environment variables: -e MP_JWT_VERIFY_PUBLICKEY="..." and -e MP_JWT_VERIFY_ISSUER="..."

2. Download a sample

Welcome Letter

Simple template with field replacement.

welcome.odt welcome.json
Invoice

Template with loops and calculated fields.

invoice.odt invoice.json

3. Generate a PDF

curl -X POST http://localhost:8080/api/render/template/upload \ ①
  -H "Authorization: Bearer eyJhbG...R37F6g" \ ②
  -H "Accept: application/pdf" \ ③
  -F "template=@welcome.odt" \ ④
  -F "data=@welcome.json" \ ⑤
  -o welcome.pdf ⑥
  1. API endpoint for multipart template upload
  2. Dev JWT token (included for quick testing). Get the token
  3. Desired output format (application/pdf → PDF, application/rtf → RTF, or application/vnd.oasis.opendocument.text → ODT)
  4. ODT template file — see the README for authoring details.
  5. JSON data file with template values
  6. Save generated document to file
